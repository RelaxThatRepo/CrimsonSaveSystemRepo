Crimson Save System

Welcome to the documentation for the Crimson Save System, a modular, fragment-based save solution for Unreal Engine 5. This system is designed to be highly decoupled, allowing any number of game systems to register themselves as "saveable" without creating hard dependencies.

It intelligently separates persistent player data (like inventory and skills) from level-specific world data (like open chests or destroyed objects).
