Installation Guide

There are two primary ways to install the Crimson Save System, depending on how you acquired it. Follow the method that matches your situation.

Method 1: Epic Games Launcher (Marketplace)

This is the standard method if you acquired the plugin from the Unreal Engine Marketplace. This method is compatible with both Blueprint-only and C++ projects.

  1. Open the Epic Games Launcher.
  2. Navigate to your Library page.
  3. Scroll down to your "Vault" section and find the Crimson Save System plugin.
  4. Click the "Add to Project" button.
  5. A new window will appear. Select the project you want to add the plugin to. Make sure the engine version is compatible.
  6. Click "Add". The launcher will copy the plugin files into your project's Plugins folder.
  7. Open your Unreal Engine project.
  8. You should get a pop-up in the bottom-right corner: "New plugins are available." Click "Manage Plugins...".
  9. If you don't see the pop-up, go to Edit > Plugins in the main toolbar.
  10. Find "Crimson Save System" (it's likely under the "Project" or "Installed" category).
  11. Check the "Enabled" box.
  12. The editor will prompt you to restart. Click "Restart Now".

If your project is C++ based, the editor may ask you to rebuild the project. This is normal. Close the editor, open your IDE (Visual Studio/Rider), and build the project. Then, re-open the editor.

Method 2: Manual C++ (Source) Installation

This method is for C++ projects when you have the plugin's source code (e.g., from GitHub or a direct download).

  1. Download the plugin source code, which is usually a .zip file.
  2. Navigate to your project's root directory (the folder containing your .uproject file).
  3. Create a new folder named Plugins if one does not already exist.
  4. Unzip the plugin files. You should have a folder named CrimsonSaveSystem (or similar).
  5. Move the entire CrimsonSaveSystem folder into the Plugins folder.
  6. The final directory structure should look like this:
    7. MyProject/
├── Content/
├── Source/
├── Plugins/
│   └── CrimsonSaveSystem/
│       ├── Source/
│       ├── Resources/
│       └── CrimsonSaveSystem.uplugin
└── MyProject.uproject
  7. Open your .uproject file in a text editor (like Notepad or VS Code). Find the "Plugins" array (or create it if it doesn't exist) and add the following entry. This ensures the project enables the plugin on load.
    8. {
    ...
    "Plugins": [
        {
            "Name": "CrimsonSaveSystem",
            "Enabled": true
        }
        // ... any other plugins you have
    ]
}
  8. Save and close the .uproject file.
  9. Close your Unreal Editor and IDE (Visual Studio, Rider) if they are open.
  10. Right-click on your project's .uproject file and select "Generate Visual Studio project files".
  11. Once complete, open your project's .sln file in your IDE.
  12. Build the project (e.g., press Ctrl+Shift+B in Visual Studio). This will compile the plugin's source code along with your game.
  13. After the build succeeds, you can open your project in the Unreal Editor. The plugin will be enabled.

Verification

You can confirm the installation was successful in two ways:

  • Go to Edit > Plugins and verify that "Crimson Save System" is listed and marked as "Enabled".
  • In a Blueprint, right-click and search for "Get Crimson Save Game Manager". If the node appears, the subsystem is loaded and ready.
